# 📊 Case Churn — Análise Exploratória de Dados (EDA)

Análise exploratória de dados para validar a hipótese de aumento na taxa de churn de uma plataforma de ERP voltada para pequenas e médias empresas.

> 📚 Dados obtidos no curso [EBA Analista](https://renatabiaggi.com/eba-analista/) da Renata Biaggi.

---

## 🎯 Objetivo

Validar a hipótese de aumento na taxa de churn identificando padrões e comportamentos que possam apoiar a equipe de negócios na tomada de decisão.
